# M & G Tax Services LLC

Professional tax preparation and bookkeeping for individuals and small businesses. Miami, FL.

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS
- TypeScript

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
pnpm build
pnpm start
```
